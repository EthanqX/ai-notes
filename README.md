## Running Locally

### Prerequisites
start docker
```bash
$ cd local-development
$ docker-compose up -d

```

### run
```bash
$ ./gradlew bootRun
```


### Formatter
```bash
$ ./gradlew spotlessApply
```

### style check
```bash 
$ ./gradlew checkstyleMain
```

### build
```bash
$ ./gradlew build
```
